# Science.Cryptography.Ciphers

This project collects classic ciphers implemented in .NET/C#.

Accepting contributions for the following ciphers:

* ADFGVX
* Columnar Transposition 
* Enigma
* Hill
* Permutation
* Playfair
* Rail fence
* Myszkowski Transposition
* Nihilist
* One Time Pad
* Solitaire
* Trifid

* Any other missing :)
